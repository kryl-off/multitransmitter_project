Отправлять только отчеты не получиться, так как есть импульсные датчики двиения, которые отправляют импульс на короткое вермя и можно его пропустить. Ну и дверь если быстро открыть.

Структура пакета
1 байт id устройства (привязка по ip потребует дополнительных действий, а так будет отдельный идентификатор)
1 байт тип пакета (1 - событие)
	1 - событие датчика
	2 - отчёт
	3 - ping
	4 - событие устройства


Для события датчика
	4 Байта временная метка
	1 Байт id датчика
	1 Байт тип события
		0 - Открытие двери
		1 - Закрытие двери
		2 - Сильная смена температуры
		3 - Датчик движения
		...

	X Байт Дополнительные данные в зависимости от типа события
		2 байта - новая температура
		...

Для ежеминутных отчетов (ежесекундных) (такой набор для каджого датчика в одном пакете)
	? Временная метка
	1 Байт id датчика
	1 Байт тип датчика
		0 - Дверной датчик
		1 - Температурный датчик
		...
	X Байт Данные в зависимости от типа датчика
		1 байт - 0/1 Закрыто/Открыто для дверного датчика
		2 байта - температура
		...

Для события устройства
	1 Байт тип события
		0 - Время получено
		1 - Настройки применены
		2 - Возврат настроек
		3 - Датчик движения
		...

	X Байт Дополнительные данные в зависимости от типа события
		4 байта - Тип настроки + текущие данные.
		...
